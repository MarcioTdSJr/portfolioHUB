# 🛡️ portfolioHUB

Bem-vindo ao meu repositório central! Aqui documento minha jornada acadêmica e profissional, com foco no desenvolvimento de software seguro, infraestrutura de redes e análise de dados.

## 📂 Estrutura do Repositório

Este repositório está organizado nas seguintes seções:

* **`/Projetos_Academicos`**: Aplicações desenvolvidas em Java com foco em Princípios SOLID e Clean Architecture.
* **`/Infraestrutura_e_Seguranca`**: Simulações de roteamento e segmentação de redes criadas no Cisco Packet Tracer.
* **`/Projetos_Profissionais`**: Documentação de painéis interativos de Business Intelligence (Power BI) desenvolvidos para otimização de métricas estratégicas.

## 💻 Tecnologias e Ferramentas
* **Segurança e Redes:** Princípios de Cibersegurança, Cisco Packet Tracer.
* **Desenvolvimento:** Java, Modelagem UML, Orientação a Objetos.
* **Dados:** Power BI.
